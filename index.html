How this works

Data pipeline

A GitHub Action pulls QQQ and SPY daily opens from Yahoo Finance at 9:30, 9:35, 9:40, and 9:45 ET every market day, computes the indicators, and commits data/latest.json.

Manual refresh

Press Refresh to reload the latest committed data. To force a fresh Yahoo pull, run the Update dashboard data workflow from the Actions tab in GitHub (Run workflow).

Reading the cards

Each card shows whether today's open should trigger a buy. Green rows satisfy the rule; red rows don't. A card is BUY only when every required condition passes.

Signal source: Yahoo Finance via yfinance. Not investment advice — use your own judgement.
