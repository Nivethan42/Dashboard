HOW THIS WORKS
Data pipeline

A GitHub Action pulls QQQ and SPY daily opens from Yahoo Finance at 9:30, 9:35, 9:40, and 9:45 ET every market day, computes every indicator, and commits data/latest.json.

Manual refresh

Press Refresh to reload the latest committed data without CDN caching. To force a fresh Yahoo Finance pull, run Update dashboard data from the GitHub Actions tab.

Reading a card

Each row: dot (pass/fail) · rule with threshold · today's value coloured green or red. The ANY group needs at least one to fire; the ALL group needs every rule to hold. Both must be true for a BUY.

Signal source: Yahoo Finance via yfinance. Not investment advice.
